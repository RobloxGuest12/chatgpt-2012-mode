# ChatGPT 2012 Mode

[Click here to Install The Style](https://robloxguest12.github.io/chatgpt-2012-mode/chatgpt-2012-mode.user.css) or [Here](https://userstyles.world/style/26931/chatgpt-2012-mode) to get it on userstyles.world

Make ChatGPT look like a **simple early‑2010s website** instead of the
modern rounded UI.

This style removes modern glass effects and brings back **classic
gradients, square corners, and simple forum-style messages** --- similar
to browsing the web in **2012**.

![Preview](https://userstyles.world/preview/26931/1.webp)

## Features

* Classic gradient backgrounds like older websites
* Square UI corners instead of modern rounded ones
* Removes blur / glass effects
* Forum-style chat message boxes
* User messages highlighted with light blue
* Assistant messages styled with simple grey panels
* Old-style gradient buttons and hover effects
* Compact layout with reduced spacing

## What it changes

This userstyle modifies:

* Chat message containers
* Sidebar appearance
* Input box styling
* Button styling and hover effects
* Page background
* Layout width and spacing

The goal is to recreate the feeling of **early 2010s web design**.

## Supported Sites

* [https://chatgpt.com](https://chatgpt.com)
* [https://chat.openai.com](https://chat.openai.com)

## Notes

* Works best with the **Stylus** browser extension
* Uses `@-moz-document` for domain targeting
* Designed for a nostalgic UI experience rather than a perfect
  historical recreation
* Only Works on Light Mode
* Requires [Stylus](https://github.com/openstyles/stylus)  

## Inspiration

Inspired by the design style commonly seen on:

* Early web applications
* Classic forums
* Gradient-heavy UI design from around **2010--2013**
